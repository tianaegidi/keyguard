Workers API

Receives only encrypted payloads and handles create, claim, and expiration requests at the edge.

DO

Durable Objects

Provides atomic one-time claiming, ciphertext storage, and automatic expiration so two viewers cannot reveal the same secret.

T

Turnstile + Rate Limiting

Protects link creation and reveal endpoints from bots and automated abuse.

O

Workers Observability

Tracks reliability and failures while deliberately excluding keys, ciphertext, and passwords from logs.
